# Usage
<!-- usage -->
```sh-session
$ npm install -g voyant-cli
$ voyant-cli COMMAND
running command...
$ voyant-cli (-v|--version|version)
voyant-cli/0.0.0 darwin-x64 node-v12.16.1
$ voyant-cli --help [COMMAND]
USAGE
  $ voyant-cli COMMAND
...
```
